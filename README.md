# PlanetEarth 가이드

환영합니다! PlanetEarth 가이드 저장소입니다.


## 로컬 실행

Node.js 22+ 와 pnpm 이 필요합니다.

```bash
pnpm install
pnpm start
```

## 빌드 및 검증

```bash
pnpm build
pnpm typecheck 
```
