## 플롯(Plot) 이란

마인크래프트의 월드 단위인 청크(16*16칸 안의 모든 블럭)와 1대1로 대응되는 타우니에서의 땅 단위입니다.

![plot](/img/guide/towny/plot-permissions/plot-permissions-plot-01.webp)

흔히들 말하는 마을 사이즈가 N 이다 에서 이 N이 마을이 가지고 있는 플롯의 숫자 입니다. 위 사진에서는 Town Size가 153이므로, 해당 마을은 153 플롯(청크)를 자신의 땅으로 가지고 있다고 알려줍니다.

플롯에 대한 권한이 어떻게 설정되어 있는지 보려면 **/plot perm hud** 명령어로 UI를 통해서 알 수 있습니다.

## 플롯 권한 설정이 마을 권한 설정과 다른 점

플롯 권한 설정과 마을 권한 설정 중 가장 큰 차이점은, 권한을 부여할 수 있는 단위가 다르다는 것 입니다.

![section](/img/guide/towny/plot-permissions/plot-permissions-section-01.webp)

Plot 권한 설정 HUD

![section](/img/guide/towny/plot-permissions/plot-permissions-section-02.webp)

Town 권한 설정 HUD

좌측의 플롯 권한 설정은 f, t, a, o집단에게 권한을 줄 수 있고, 우측의 마을 권한 설정은 r, n, a, o집단에 권한을 줄 수 있습니다. 

여기에서 “집단” 이란 그 땅의 주인과, 나머지 유저들 간의 “관계”를 의미합니다. 마을 공용 땅이 아닌 자신의 땅일 경우 f는 그 땅 주인의 “친구”, t는 그 땅 주인과 “같은 마을원 혹은 같은 국가원”을 의미합니다. 그러면 각각의 권한 설정에 대해서 상세하게 설명 해 드리겠습니다.

## Plot 권한 설정

Plot 권한 설정은 두가지 방식이 있습니다.

- 첫번째로, 마을원에게 플롯의 소유권을 주지 않고 시장이나 부시장이 “직접” 관리하는 방법입니다.
해당 방식은 시장이나 부시장이 마을 청크 위에 가서 플롯 설정 명령어를 이용하는 방식입니다.
주로 마을 공용 창고, 국가 공용 창고, 공용 농장 등 공적인 부분에 사용하는 것을 권장드립니다.
- 두번째로, 마을원에게 플롯의 소유권을 줘, 해당 플롯의 주인이 알아서 관리하게 하는 방식입니다.
해당 방식은 마을원에게 해당 플롯의 자치권을 주고, 받은 마을원이 해당 플롯 위에서 명령어를 사용하여 설정하는 방식입니다. 주로 개인 건축물, 개인 창고 등에 사용하는 것을 권장드립니다.

### Plot 권한 설정 명령어

:::info
**/plot set perm [resident/nation/ally/outsider] [build/destroy/switch/itemuse] [on/off]**
서있는 플롯의 권한을 설정하는 명령어 입니다.
:::

:::info
/plot trust add [닉네임]
플롯에 집단 단위가 아닌 특정 유저에게 권한을 주는 명령어입니다. (마을 Trust와 비슷함)
:::

:::info
/plot fs [금액] (기본값 0)
플롯을 마을원이 설정된 금액을 지불하고, 해당 플롯의 소유권을 가져갈 수 있는 상태로 만듭니다.
:::

:::info
/plot claim
마을원이 [/plot fs ~]로 취득 가능한 플롯을 금액을 지불하고 취득합니다.
:::

:::info
/plot info
서있는 플롯의 자세한 정보를 확인합니다.
:::

**사용 예시**

- 내가 서있는 플롯에서 마을원은 블럭 설치만 가능해 → /plot set perm resident off (해당 플롯에서 마을원의 전체 권한 제거) → /plot set perm resident build on (해당 플롯에서 마을원의 설치 권한 지급)
- 내가 서있는 플롯에서 국가원은 상자 이용만 가능해 → /plot set perm nation off (해당 플롯에서 국가원의 전체 권한 제거) → /plot set perm nation switch on (해당 플롯에서 국가원에게 Switch 권한을 지급하여 상자 이용을 가능하게 만듬)
- 특정 Plot에 집단이 아닌, “특정 유저” 에게만 모든 권한을 주고 싶어 → /plot set perm
- (마을 시장) 플롯을 다른 마을원이 30골드에 취득했으면 좋겠어
    - /plot fs 30
    - /plot claim (마을원이 명령어 실행)
    - (마을원) 내가 얻은 플롯에서 나와 친구로 되어있는 유저만 권한을 주고 싶어
        - /plot set perm off → /plot set perm friend on
- (마을 시장) 누군가 점유중인 플롯의 점유를 해제시키고 싶어 → 해당 청크로 이동 후 /plot evicit
- 내가 서있는 플롯에서 나 말고는 아무도 무언가를 할 수 없게 하고 싶어 → /plot set perm off

## 마을 권한 설정

마을 권한 설정은, 아무런 플롯 권한 설정이 되어있지 않은 플롯에만 적용이 되며, 소유주가 있거나, 따로 설정이 되어 있는 플롯의 경우 적용되지 않습니다.

### 마을 권한 설정 명령어

:::info
**/t set perm [resident/nation/ally/outsider] [build/destroy/switch/itemuse] [on/off]**
마을 전체 플롯의 권한을 설정하는 명령어 입니다.
:::

:::info
/t trust add [닉네임]
특정 유저에게 마을의 권한을 모두 줍니다. (남용하지 않는 것을 권장합니다.)
:::

**사용 예시**

- 우리 마을은 국가원이 모든 권한을 자유롭게 사용했으면 해 → /t set perm nation on (국가원에게 BUILD, DESTROY, SWITCH, ITEMUSE 권한을 모두 허용)
- 우리 마을은 국가원이 상자만 열 수 있게 하고 싶어 → /t set perm nation off (국가원의 모든 권한 제거) → /t set perm nation switch on (국가원에게 SWITCH 권한만 부여)
- 우리 마을은 동맹원도 상자를 열 수 있게 하고 싶어 → /t set perm ally off (동맹의 모든 권한 제거) → /t set perm ally switch on (동맹에게 SWITCH 권한만 부여)

## Plot 그룹 권한 설정 (중요)

앞에서의 플롯 설정은 한가지 단점이 있습니다. 모든 청크마다 따로 따로 설정을 해줘야 합니다. 따라서, 여러번의 반복 작업이 모든 유저에게 부과됩니다. 이 문제점을 해결하기 위하여 Plot Group이라는 기능이 타우니에는 존재합니다.

![plot](/img/guide/towny/plot-permissions/plot-permissions-plot-02.webp)

Plot 그룹 기능을 사용하면, [/plot perm hud] 명령어를 사용하였을 때 맨 윗줄에 Plot Group 라는 항목이 생깁니다.

## Plot 그룹 권한 설정 명령어

### 기초

:::info
/plot group new [그룹명]
[그룹명] 이라는 Plot 그룹을 만들고, 서있는 플롯을 만드는 그룹에 포함시킵니다.
:::

:::info
/plot group add [그룹명]
서있는 플롯이 아무 그룹에 속해있지 않다면, [그룹명] 그룹에 소속시킵니다.
:::

:::info
/plot group remove
서있는 플롯을 그룹에서 제거합니다.
:::

:::info
/plot group delete
위치해 있는 그룹을 완전히 삭제합니다.
:::

:::info
/plot group rename [이름]
위치해 있는 그룹의 이름을 [이름]으로 변경합니다.
:::

### 거래

플롯 그룹도 기존 플롯처럼 마을원 간 거래가 가능합니다.

:::info
/plot group fs [금액]
마을원이 [금액]을 지불하여 위치해 있는 그룹을 취득할 수 있게 합니다.
:::

:::info
/plot group notforsale
위치해 있는 그룹을 취득할 수 있는 상태에서 불가능한 상태로 변경합니다.
:::

### 권한

:::info
**/plot group set perm [friend/town/ally/outsider] [build/destroy/switch/itemuse] [on/off]**
플롯 그룹의 권한을 설정하는 명령어 입니다.
:::

**사용 예시**

- 3개의 Plot을 Dev라는 그룹에 속하게 하고 다른 마을원이 이 그룹을 0골드로 취득했으면 좋겠어
    - /plot group new Dev → /plot group add Dev (나머지 두 청크에서 각각 실행) → /plot group fs 0
    - (그룹을 취득한 마을원) 이 그룹은 이제 내꺼야, 이 그룹 안에서 나와 친구가 되어 있는 유저들만 자유롭게 활동했으면 해
        - /plot group set perm off (본인 외 모든 유저의 권한 제거) → /plot group set perm friend on (본인과 친구 상태인 유저에게만 모든 권한 허용)
- 평소에는 마을원과 국가원이 사용할 수 있지만, 전쟁때는 동맹원까지 사용 가능한 공용 창고를 만들고 싶어
    - /plot group new storage → /plot group add storage (처음 NEW 명령어를 친 플롯을 제외하고 원하는 플롯을 다 넣을 때 까지 반복) → /plot group set perm town switch on (마을원, 국가원은 상자 사용이 가능한 상태)
    - (전쟁 시작 전) /plot group set perm ally switch on (동맹원들에게 이 그룹 안에 속해있는 청크에서 SWITCH 권한 부여)
    - (전쟁 종료 후) /plot group set perm ally switch off (동맹원들이 더이상 해당 그룹에 속해있는 영역의 상자를 사용하지 못함)
