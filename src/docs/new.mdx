# 👋서버에 처음 왔어요!👋

서버에 처음 오셨군요! 앞으로 보여드릴 가이드를 잘 기억해주세요

서버에 처음 접속하시면 기본템과 함께 서버 스폰에서 시작합니다. 
서버 스폰에서는 추후 플레이 하실 여러 컨텐츠 관련 상점 및 포탈을 이용할 수 있습니다.

우선, 인벤토리에 들어있는 플래닛어스 가이드북을 우클릭하여 읽어보세요!

![overview](/img/guide/meta/new/new-overview-01.webp)

:::caution
❗ **뉴비 분들은 서버 플레이타임을 기준으로 3시간 동안 PVP로부터 보호받습니다.**
:::

## 🐥마을에 가입하세요!🐥

플래닛어스는 **마을과 국가에 소속되어있어야 정상적인 플레이가 가능**합니다. 

**마을/국가 설립 및 운영은 뉴비 혼자서 하기엔 꽤 어렵습니다.** 혼자 오신 뉴비분들은 다른 마을에 가입하는 것을 적극 추천드립니다. **가이드북을 이용하여 마을 가입 요청을 보내세요!**

![section](/img/guide/meta/new/new-section-01.webp)

또한 **서버 공식 디스코드의 국가 홍보 채널**에 올라온 국가 디스코드 서버에 들어가서 나라를 구할 수 있습니다. 많은 국가가 새 국민을 받고 있으니 꼭! 확인해보세요.

만약 다른 마을에 가입하지 않고 함께 플레이 할 사람들과 국가를 세우려는 분들은
  [타우니 기초 가이드](/commands) 를 확인하세요.

## 🙄앞으로 무엇을 해야 할까요?🙄

- [ ]  /t spawn 명령어로 가입한 마을로 이동하세요!
- [ ]  초대한 마을장과 마을에서 만나 정착을 시작하세요
- [ ]  타우니 관련 유용한 명령어 및 가이드를 확인하세요.
 [타우니 가이드/명령어](/commands)
- [ ]  서버 규칙을 확인했나요? 확인하지 않았다면 서버 규칙을 꼭! 확인해 주세요.
 [서버 규정](/rules)
- [ ]  다른 가이드들도 읽어 보셨나요? 더 편하게 플레이 하실 수 있어요!
Shift + F를 누르거나 /menu 명령어로 메인 메뉴를 연 뒤, 가이드 버튼을 클릭하세요.

![section](/img/guide/meta/new/new-section-02.webp)

```jsx
**BATTLEPASS** - 배틀패스 메뉴 오픈
**QUEST** - 퀘스트 메뉴 오픈
**UPGRADE** - 강화 메뉴 오픈
**GUIDE** - **가이드** 메뉴 오픈
**지도**(GUIDE 오른쪽) - 웹 지도 주소 출력
**망치와 톱**(GUIDE 오른쪽, 지도 밑) - 커스텀 조합법 오픈
**TRADE** - **유저 거래소** 오픈
**POINT SHOP** - **포인트 상점** 오픈
**DAILY** - **출석체크** 메뉴 오픈
**SETTING** - 개인 설정 메뉴 오픈
**보석** (SETTING 밑) - **특산품** 남은 시간 확인
**WARP** - 워프(스폰, 홈 등) 관련 도움말 출력
**열쇠**(WARP 밑) - 추천 링크 출력 및 누적된 열쇠 획득
**MARKET** - **서버 상점** 오픈
**CLASS** - **병과** 선택 메뉴 오픈
**FISH** - 커스텀 물고기 판매 메뉴 오픈
**DUEL** - 듀얼 메뉴 오픈
**엔더상자** - 엔더상자 오픈(랭크 구입 후 이용 가능)
**조합대** - 조합대 오픈(랭크 구입 후 이용 가능)
```

## [**💵**](https://emojipedia.org/dollar-banknote/)경제 활동[**💵**](https://emojipedia.org/dollar-banknote/)

플래닛어스에서는 여러가지 방법으로 돈을 벌 수 있습니다!
술 제작(양조), 광질, 농사, 목축 등 여러가지 방법으로 돈을 벌 수 있습니다.

서버를 시작할 때 기본템으로 지급받은 에메랄드와 밀을 팔아봅시다!

- Shift (웅크리기) + F(손 바꾸기) 를 눌러 서버 상점을 연 뒤, 에메랄드와 밀을 팔아보세요.
