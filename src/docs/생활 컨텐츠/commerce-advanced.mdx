## **상단 버프**

무역선과 당나귀를 타고 이동 중에 주변 40블록 내에 무역협정을 체결한 국가의 국민이 많을수록 주기적으로 보급품이 감소하지 않을 확률이 증가합니다.

| **상단 인원 (본인 포함)** | **주기적으로 보급품 감소하지 않을 확률** |
| --- | --- |
| 1명 | 0% (미적용) |
| 2명 | 12% |
| 3명 | 15% |
| 4명 | 18% |
| 5명 | 21% |
| 6명 이상 | 24% |

## **운하**

세계 곳곳에는 무역선이 통과할 수 있는 운하가 존재합니다.
운하를 이용하면 **바다를 우회하지 않고 무역 이동 시간을 크게 단축**시킬 수 있고,
**무역선 ↔ 당나귀 환승보다 더 저렴한 비용**으로 육지를 통과할 수 있습니다.

운하는 NPC 마을로 점유되어 있고, 매일 오후 11시에 운하 마을과 인접한 국가 중 가장 많은 영토를 점유중인 국가는 해당 운하를 점유합니다.

운하를 통과하려면 운하를 점유중인 국가와 무역협정이 체결되어야 하고, 
운하 중심에 있는 색유리를 바라본 상태에서 `/무역 운하통행` 명령어를 2번 사용하면
일정 골드를 지불하고 운하를 통행할 수 있습니다.

## 사건

육로 이동 또는 항해 중에는 여러 지역에서 다양한 사건들이 일어납니다.

사건에 잘 대처하면 이로운 효과를 얻을 수 있고, 반대로 사건에 대처하지 못하면 패널티를 받습니다.

현재 사건은 베타 테스트 중입니다. 추후 더 많은 사건들이 추가됩니다.

(발생 지역 지도는 100% 정확하지 않으며, 실제 좌표와 약간의 오차가 있을 수 있습니다.)

| **사건** | **설명** | **발생 지역** |
| --- | --- | --- |
| **바르바리 해적** | 바르바리 해적이 지중해의 상선을 약탈합니다.<br/>지중해를 항해중인 무역선은 60초 내에 무역협정을 체결한 국가의 영해로 대피해야 합니다.<br/><br/>대피 성공 시: 보급품 감소 안함<br/>대피 실패 시: 보급품 10 감소<br/>사건 발생 시 인벤토리 핫바에 내구도 30이상 함포 보유 시: 대피 불필요, 보급품 2 획득, 함포 내구도 30 감소 | 지중해<br/>`-720 < X < 2600`<br/>`-5900 < Z < -4100` |
| **강우** | 월드에 비가 내리기 시작할때, 항해중인 모든 무역선의 보급품이 2 증가합니다. |  |
