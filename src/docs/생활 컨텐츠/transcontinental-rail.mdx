## 이용 방법

- 아래 사진의 표지판에서 원하는 방향을 우클릭 하세요.
- 표지판 우클릭 시 다음 역으로 이동하기까지 1분의 쿨타임이 부여됩니다.
남은 쿨타임은 표지판을 재클릭 시 확인할 수 있으며, 쿨타임 종료 시에 알림 메세지가 뜹니다.
- 서버 재시작 후 최초 이용 시 1분의 쿨타임이 부여됩니다.
- 마지막 사용이 5분 이상 지난 경우에도, 서버 재시작 처럼 1분의 쿨타임이 부여됩니다.
- 1개의 정거장 당 1분의 이동 소요 시간과, 일정량의 골드가 부과됩니다. 
ex) 정거장 5개 이동 시 5분 + 골드 5회 사용
- 전쟁 중에는 쿨타임이 3분 30초로 늘어나니 이용에 유의해 주세요.
