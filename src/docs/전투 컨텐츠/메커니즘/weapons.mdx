# 화기: 머스킷, 피스톨, 대포

## 머스킷

플래닛어스에는 머스킷이 존재합니다. 머스킷은 활이나 석궁보다 장전 시간이 훨씬 길고 사거리가 짧은 원거리 무기이지만, 즉발로 높은 고정 데미지를 넣을 수 있다는 장점이 있습니다. 

머스킷은 머스킷병으로만 사용 가능하며, 인벤토리에 다른 무기가 있을 시 사용 불가능합니다. 또한 말을 탄 상태로 머스킷을 쏠 수 없습니다.

- 재장전 시간: 4초
- 데미지: 3.2칸의 고정 데미지 (헤드샷 시 3.5칸)
    - 머스킷 총알에 맞은 플레이어는 4초동안 속도 감소 1, 나약함 1의 디버프를 받습니다.
    - ~~방패로 막을 시 70%의 데미지를 받습니다.~~
    - 피격당한 사람의 네더라이트 갑옷 파츠 수에 비례하여 데미지가 줄어듭니다.
    - 피격당한 사람이 갑옷을 입고있지 않으면 갑옷 파츠 수에 비례하여 데미지가 증가합니다.
- 총검: 머스킷을 들고 적을 근접 공격하면 총검으로 공격하여 다이아검 만큼의 데미지를 입힙니다. 발사 후 2초 동안은 총검 사용이 불가능합니다.

실제 머스킷처럼 칼, 활에 비해 배우기 쉽고 사용법도 간단하여, 전쟁 시 많은 인원을 동원해야 할 때 주력 무기로 자주 사용됩니다.

![section](/img/guide/pvp/weapons/weapons-section-01.webp)

## 피스톨

피스톨은 머스킷과 시대적 배경을 공유하는 초창기 화기이며, 플린트락 방식으로 머스킷과 동일하게 단발 장전 및 발사됩니다. 피스톨은 모든 병과에서 사용할 수 있는 **보조 무기**입니다. 모든 병종이 사용할 수 있는 보조 무기인 만큼 재장전 시간이 길지만, **고증처럼 미리 장전된 피스톨 여러 개를 가지고 다니면서 사용할 수 있습니다.**

- 재장전 시간: 6초
- 데미지: 2.1칸의 고정 데미지
    - ~~방패로 막을 시 70%의 데미지를 받습니다.~~
    - 피격당한 사람의 네더라이트 갑옷 파츠 수에 비례하여 데미지가 줄어듭니다.
- 근접병은 1개, 궁병은 2개, 머스킷병은 3개의 피스톨을 인벤토리에 들고 다닐 수 있습니다.
- 모든 병과가 사용 가능합니다
- 말을 탄 상태로 쏠 수 있습니다.
- 머스킷보다 상대적으로 낮은 명중률과 총알 속도를 가지지만, 오른손에 들고 있어도 느려지지 않습니다. 또한 머스킷과 같은 총알을 공유합니다.

모든 총 발사에는 2초의 글로벌 쿨타임이 적용됩니다 (순식간에 여러 개의 총을 번갈아 쓰며 연사하는 행위 방지)

![section](/img/guide/pvp/weapons/weapons-section-02.webp)

|  | 머스킷 | 피스톨 |
| --- | --- | --- |
| 재장전 시간 | **4초** | 6초 |
| 고정 데미지 | **3.5~4.0** | 2.5~3.0 |
| 피격당한 적에게 디버프 적용 | **적용** | 미적용 |
| 명중률 | **높음** | 낮음 |
| 총알 속도 | **빠름** | 느림 |
| 총검 사용 | **가능** | 불가능 |
| 말을 탄 상태로 사용 | 불가능 | **가능** |
| 들고 있을 때 이동속도 감소 | 감소 | **감소하지 않음** |
| 사용 가능한 병과 | 머스킷병 | **모든 병과가 보조 무기로 사용 가능** |

## 대포

마을 땅 안에 있는 블럭들은 폭발로 파괴되지만, 일정 시간이 지나면 자동으로 복구됩니다. 전쟁 중에 공격측은 방어측 마을 땅 주변 야생에 TNT를 설치하거나, 바닐라식 TNT 대포를 만들어서 적의 성벽을 부수고 마을 내부로 진입하거나 폭발로 적에게 데미지를 줄 수 있습니다.

:::info
문, 철문, 감압판, 버튼, 레버 등 옆에 블록이 파괴되면 같이 파괴되는 블록 또는 2칸 높이의 블록은 폭발복구 진행순서에 따라서 원래 블록 상태와 동일하게 복구되지 않거나, 일부만 복구되거나, 복구가 누락될 수 있습니다. 이는 의도된 기능이며 서버측에서 이로 인한 피해는 복구해드리지 않습니다.
:::

### 철 대포

철 대포는 직사포로, 정면에 있는 적을 공격하기 좋은 형태입니다. 철블럭 2개와 돌 버튼 1개 그리고 횃불 한개가 필요합니다. 박격포와 비교하여 열 관리가 조금 더 쉽습니다.

![section](/img/guide/pvp/weapons/weapons-section-03.webp)

### 박격포

박격포는 벽 넘어에 있는 적을 공격하거나, 벽 뒤에서 숨어서 적을 공격할 수 있는 형태의 대포입니다. 열 관리가 어려워 철 대포보다는 연사력이 떨어지지만, 여러 지형에 맞춰 다양하게 활용하기 좋습니다.

![section](/img/guide/pvp/weapons/weapons-section-04.webp)

- 대포를 전쟁시간 외 다른나라에 사용하는 것은 처벌됩니다.
- 대포로는 마을 땅 안의 블럭만 파괴할 수 있습니다.
