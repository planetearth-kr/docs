# 컨퀘스트(Conquest)

### **점령지 근처로 텔레포트**

컨퀘스트에 참여하려면 우선 점령지 근처로 가야 하며, 시작하면 컨퀘스트 텔레포트 메뉴가 활성화됩니다.
컨퀘스트 점령지는 추가, 삭제, 리메이크가 될 수 있습니다.

### 컨**퀘스트 진행**

일반 점령전과 달리, 컨퀘스트는 **점령지가 총 7개 혹은 8개가 존재**합니다.

컨퀘스트는 **45분 동안 진행**되며, 국가가 점령지를 점령하면 **20초마다 일정 점수를 획득**합니다.

**45분이 지나서 컨퀘스트가 종료되면, 획득한 점수만큼의 컨퀘스트 열쇠 조각**을 얻습니다. 

컨퀘스트 열쇠 조각은 스폰의 상인에게 점령전 열쇠로 교환 가능합니다.

- 각 점령지는 신호기의 색 혹은 하늘에 떠있는 색유리로 구분이 가능합니다.
- **/conquest** 명령어로 현재 진행 중인 컨퀘스트 정보와 국가별 점수를 확인할 수 있습니다.
