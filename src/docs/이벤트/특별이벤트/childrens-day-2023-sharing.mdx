# 어린이날 이벤트 - 나눔

[진행기간] 2023/05/02 ~ 2023/05/05

[진행내용]

**그래픽카드 & 키보드 나눔 이벤트**
저희 서버 대표 스트리머인 권유랑님과 협업하여 그래픽카드 나눔이라는 좋은 기회를 마련하게 됐습니다. GTX1050이 좋은 그래픽카드는 아니지만, 
이거라도 있어야 뭐라도 할수 있다 하는 어린이 유저들은

[https://cafe.naver.com/kwonyurangdan](https://cafe.naver.com/kwonyurangdan)

권유랑님 네이버 카페에 사연을 올려주시면 사연 검토 후 택배 배송을 해드리겠습니다.
사연을 작성하실 때 권유랑님의 방송에 해당 사연이 송출될 예정이므로, 자신이 누군지 특정지을 수 있는 내용은 제거해주세요. (ex-> 저는 xxx초등학교에 다니는, 닉네임 xxx 입니다, 저는 xx에 사는 xxx입니다.)
