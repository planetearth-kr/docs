# 어린이날 이벤트 - 보물찾기1

[진행기간]  2023/05/05

[진행내용]

어릴 때 소풍을 가면 항상 하던 보물찾기, 플래닛어스에서 즐겨보세요!
이벤트 맵 곳곳에 숨겨진 상자를 찾아 보물을 찾을 수 있습니다.

보물에는 인게임 재화, 뚜레쥬르 기프티콘 등 소정의 상품이 포함되어 있습니다.
