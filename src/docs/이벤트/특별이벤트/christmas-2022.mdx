# 크리스마스 이벤트

[진행기간]  2022/12/25

[진행내용]

```
 드디어 크리스마스가 다가왔습니다!
그동안의 플어 여러분의 성원에 힘입어 보물찾기 이벤트를 진행하고자 합니다.
크리스마스 한정맵에서 각종 보상이 담긴 상자를 찾아 흭득하시면 됩니다.
최대한 많은 상자를 찾으시면 더 많은 보상을 얻어가실수 있습니다.
```
