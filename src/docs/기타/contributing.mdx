# 가이드 직접 수정하기

플래닛어스 공식 가이드는 [GitHub 저장소](https://github.com/planetearth-kr/docs)에서 관리됩니다. 누구나 기여할 수 있습니다.

## 로컬에서 미리보기

```bash
pnpm install
pnpm start
```

브라우저에서 `http://localhost:3000`으로 접속하면 변경 사항을 확인할 수 있습니다.

## 기여 방법

### 1. GitHub에서 직접 수정

각 문서 페이지 하단의 **「GitHub에서 이 페이지 편집하기」** 링크를 클릭해 바로 수정할 수 있습니다.

### 2. Pull Request

1. 저장소를 Fork합니다.
2. `src/docs/` 아래 MDX 파일을 수정합니다.
3. `pnpm build`로 빌드가 성공하는지 확인합니다.
4. Pull Request를 엽니다.

### 3. Issue로 제안

오타, 내용 추가, 개선 제안은 GitHub Issue로도 접수할 수 있습니다.

- [버그 신고](https://github.com/planetearth-kr/docs/issues/new?template=1-bug.yml)
- [내용 추가](https://github.com/planetearth-kr/docs/issues/new?template=2-feature.yml)

## 작성 규칙

- 파일 형식: MDX (`src/docs/` 하위)
- 한국어로 작성합니다.
- 내부 링크는 `/slug` 형식을 사용합니다. (예: `[후원](/shop)`)
