# 플래닛어스 소개 및 뉴비 가이드

## 플래닛어스에 오신것을 환영합니다!

*"1:750 비율의 지구맵에서 펼쳐지는 또다른 세계에 몸을 던져 보세요!"*

![section](/img/guide/meta/start/start-section-01.webp)

플래닛어스는 타우니 시스템을 기반으로 실제 지구를 축소한 지구맵에서 직접 마을과 국가를 만들고 발전시켜 나가는 **오픈월드 지구서버**입니다. 저희는 무역, 건축, 양조, 전쟁(공성전), 정치, 외교 등 다양한 컨텐츠와 플레이스타일을 제공, 지향하며, 플레이어는 국가원, 국가 지도자, 군인, 상인, 건축가, 탐험가 등 여러 모습으로 서버에 참여하여 또 다른 세계에서 살아갑니다.

*플래닛어스는 첫 오픈을 한 뒤 현재까지 **초기화 없이** 같은 맵으로 진행중입니다.*

:::info
🌏 마을을 만들어 시민을 모으고 국가를 세워 국가를 통치하세요.
건축물을 만들어 국가의 위상을 높이세요.
외교로 국가에 이익을 가져오세요.
그리고 영토를 확장하고 국가를 강대국으로 만드세요.
:::
