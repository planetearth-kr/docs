## Expert 랭크

Expert 랭크는 최근 3개월 간 후원 금액이 450,000원 이상일 경우 신청할 수 있습니다.
티켓으로 최근 후원 내역을 보내주시면 심사 후 지급해드립니다.
* Expert 는 Pro 랭크와 가격이 같습니다.
* 기준 이상의 처벌 전과가 있을 경우 심사가 거절될 수 있습니다.
* 전과가 생길 경우 언제든지 랭크가 박탈될 수 있습니다.

| **혜택** |
| --- |
| 최상위 큐 우선권 |
| 원하는 칭호 설정 |
| 무기 스킨 이전 (500P) |
| 포인트 구매시 10% 보너스 |
| 전용 채팅 채널 |

후원 처리는 최대한 24시간 내에 완료하도록 노력하고 있으나, 새벽이나 이른 아침 등 담당자가 부재중인 시간대에는 처리가 지연될 수 있는 점 양해 부탁드립니다.

### 1. 계좌이체

계좌이체 완료 후, 디스코드에서 후원 티켓을 열고 입금자 이름과 함께 양식에 맞게 작성하시면 확인 후 처리됩니다.

### 2. 문화상품권
디스코드에서 후원 티켓을 열고 핀번호와 함께 양식에 맞게 작성하시면 확인 후 처리됩니다.
:::caution
문화상품권 후원은 처리가 다소 지연될 수 있습니다.

문화상품권의 경우 수수료로 인해 다른 결제 수단과는 차이가 있습니다.

랭크: 구매 기간이 한 달에 4일씩 줄어듭니다. (1개월: 30일 → 26일, 3개월: 90일 → 78일)<br />
포인트: 5,000원당 650P로 지급되는 포인트가 약간 적습니다.
:::


## 유의사항

1. 하위 랭크를 보유한 상태에서 상위 랭크를 구매할 경우 남은 랭크 기간을 새 랭크의 기간에 맞게 기간을 계산하여 연장해드립니다.
2. 후원 기간이 끝나면 추가 엔더상자 슬롯과 그곳에 있던 아이템이 사라집니다! <br />
   메인메뉴 → 포인트 상점에서 남은 기간을 확인할 수 있습니다. 끝나기 전에 추가로 랭크를 구매하거나, 엔더상자의 아이템을 빼주세요.
   만약 아이템을 제 떄 빼지 못해 아이템이 사라졌다면, 최대한 빨리 티켓에서 알려주세요. 늦으면 복구가 불가능할 수 있습니다.
3. 랭크 구매 후 서버를 **완전히 재접속** 해야 혜택이 적용됩니다.
4. 대리 구매 또한 가능합니다! 대리 구매 시 디스코드 티켓에 상품을 받는 사람의 닉네임도 함께 적어주세요.
5. 구매 후 변심에 의한 환불 또는 양도가 불가능합니다.
