# 지도

![](/img/dynmap/dynmap_preview.png)

## 지도에 접속하기

플래닛어스의 지도는 인게임에서 `/map` 명령어를 통해 받거나 [웹 지도에 직접 접속](https://planetearth.kr/map)해 볼 수 있습니다.

마우스 스크롤로 맵을 확대·축소할 수 있고, 각 버튼을 통해 지도의 다양한 기능을 사용할 수 있습니다. 지도에는 유저들이 만든 마을, 특산품, 점령전, 그리고 접속 중인 플레이어의 현재 위치가 표시됩니다.

## 기능 설명

### 좌표 보기

지도에서 특정 위치의 좌표를 확인할 수 있습니다.

<img src="/img/dynmap/dynmap_coordinates.png" className="dynmap-screenshot" alt="지도 좌표 표시" />

지도 좌측 하단에 좌표가 표시됩니다. 해당 위치는 마우스 커서 위치에 따라 변동됩니다.

좌표 왼쪽 버튼을 누르면 해당 좌표를 중심으로 하는 웹지도 링크를 클립보드에 복사합니다.

### 특정 항목을 지도에서 보거나 보지 않기

지도에서 원하는 항목만 표시하거나 숨길 수 있습니다.

<img src="/img/dynmap/dynmap_layer_button.png" className="dynmap-screenshot" alt="레이어 버튼" />

지도 좌측 상단에서 위와 같은 버튼을 클릭해주세요.

<img src="/img/dynmap/dynmap_layer_button_preview.png" className="dynmap-screenshot" alt="레이어 체크박스" />

버튼을 누르면 각 항목에 대한 체크박스가 표시됩니다. 체크박스를 토글해 원하는 항목만 볼 수 있습니다.

### 맵 타입 바꾸기

현재 보고 있는 맵을 다른 타입으로 바꿀 수 있습니다.

<img src="/img/dynmap/dynmap_menus.png" className="dynmap-screenshot" alt="지도 메뉴" />

지도 우측 상단 버튼 중 가장 왼쪽 버튼을 클릭해주세요.

<img src="/img/dynmap/dynmap_map_type_button_preview.png" className="dynmap-screenshot" alt="맵 타입 선택" /><img src="/img/dynmap/dynmap_preview_3d.png" className="dynmap-screenshot-wide" alt="3D 지도 미리보기" />

버튼을 누르면 다른 타입의 맵을 선택할 수 있습니다. 예를 들어 `world`의 `surface` 타입을 선택하면 지구 맵을 3D로 볼 수 있습니다.

### 마커 보기

특정 마을, 국가, 특산품, 전쟁 위치 등을 찾을 수 있습니다.

<img src="/img/dynmap/dynmap_menus.png" className="dynmap-screenshot" alt="지도 메뉴" />

지도 우측 상단 버튼 중 가운데 버튼을 클릭해주세요.

<img src="/img/dynmap/dynmap_marker_button_preview.png" className="dynmap-screenshot" alt="마커 카테고리" />

버튼을 누르면 각 카테고리가 표시됩니다.

<img src="/img/dynmap/dynmap_marker_button_preview_detail.png" className="dynmap-screenshot" alt="마커 상세 목록" />

원하는 카테고리를 누르면 해당 항목들이 표시됩니다. 좌표를 확인할 수 있고, 항목을 클릭하면 지도에서 해당 위치로 이동합니다. 상단 검색 칸에서 이름으로 검색할 수도 있습니다.

### 플레이어 보기

현재 접속 중인 플레이어를 볼 수 있습니다.

<img src="/img/dynmap/dynmap_menus.png" className="dynmap-screenshot" alt="지도 메뉴" />

지도 우측 상단 버튼 중 가장 오른쪽 버튼을 클릭해주세요.

<img src="/img/dynmap/dynmap_players_button_preview.png" className="dynmap-screenshot" alt="접속 플레이어 목록" />

버튼을 누르면 현재 접속 중인 플레이어가 표시됩니다. 플레이어를 클릭하면 해당 위치로 이동합니다. 상단 검색 칸에서 이름으로 검색할 수도 있습니다.

회색으로 표시된 플레이어는 현재 위치를 알 수 없다는 의미이며, 스폰에 있을 가능성이 높습니다.

## 후원자 혜택

Pro 랭크 보유자는 자신의 국가 색을 변경할 수 있습니다.

`/n set mapcolor [색깔]` 명령어로 변경할 수 있습니다. (왕만 사용 가능)

자세한 내용은 [후원](/shop) 페이지를 참고해주세요.
